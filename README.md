# 🧑‍💼 Online Examination System for Employees – Frontend Module

This project is part of an internal **Online Examination System for Employees**, developed during the **1st year, 2nd semester** as part of an academic assignment. I contributed to the **frontend development** of the platform by building key informational and interactive pages.

## 📌 Features Implemented

- **"About Us" Page** – Provides a clear overview of the platform and its purpose
- **"Contact Us" Page** – Includes form inputs for inquiries and contact details
- **"Q&A" Page** – Displays frequently asked questions with clean formatting
- Designed for **responsiveness** across desktops, tablets, and mobile devices
- Focused on **clean layout**, intuitive navigation, and user experience

## 🛠️ Tech Stack

- **HTML5**  
- **CSS3**  
- **JavaScript (Vanilla)**  
- **Development Tools**: VS Code, GitHub

## 🎯 Design Goals

- Maintain a consistent and professional look
- Ensure mobile responsiveness using flexible layout and media queries
- Create easy-to-navigate static pages with interactive elements where needed

## 📁 Project Structure Highlights

- `index.html` – Landing page
- `about.html` – About Us section
- `contact.html` – Contact form and company info
- `qa.html` – Q&A page layout
- `css/` – Stylesheets for layout and responsiveness
- `js/` – Scripts for form interaction or basic dynamic behavior
